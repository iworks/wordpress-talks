WordPress 6.5 adds AVIF support https://make.wordpress.org/core/2024/02/23/wordpress-6-5-adds-avif-support/ Q: Can WordPress create AVIF images when I upload JPEGs? Yes. Developers can use the image_editor_output_format filter to specify this type of transformation for uploads. Here is an example:

image_editor_output_format

Tworzy AVIF z pliku JPG

                        add_filter(
    'image_editor_output_format',
    function ( $formats ) {
        $formats['image/jpeg'] = 'image/avif';
        return $formats;
    }
);
