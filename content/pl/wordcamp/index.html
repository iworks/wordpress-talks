WP: eskejpuj wyjście

  • esc_html( 'Przełam Sieć' )
  • esc_url( 'http://iworks.pl/' )
  • esc_js( 'var url = "foo.bar";' )
  • esc_attr( [tablica atrybutów] )
Escape to co wypuszczasz na ekran. Zawsze! nigdy nie wiesz co ktoś/coś chce wsadzić na twoją stronę, nie wiesz co sam czynisz... dla zasady pilnuj tego. To nie tylko ochrona przed "złymi hakierami", to także ochrona przed błędami.

WP: sanityzacja

  • sanitize_title( $title )
  • intval()
  • wp_kses()
Funkcje z klasy sanitize_* pomagają w sprawdzaniu co jest czym i jakie wartości może przyjmować. Intval, absint - funkcje rzutujące na oczekiwany typ. wp_kses - wielki sprawdzacz w WP

WP: walidacja

  • biała lista
  • czarna lista
  • rzutowanie
  • podmiana
  • format
whitelist - lista możliwych wartości blacklist - lista zakazanych wartości rzutowanie - na typ (int) poadmian znaków niedopuszczonych format - sprawdzanie formatu, np. nip. /^(pl)\d{9}$/, pesel /^\d{10}$/
